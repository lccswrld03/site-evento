O QUE É UMA SELETIVA?


No universo das batalhas de rima, uma seletiva é a fase classificatória oficial que define os representantes para um torneio de maior porte.
Trata-se de uma etapa eliminatória onde os MCs disputam, com base no mérito competitivo, a oportunidade de conquistar uma vaga para avançar a campeonatos regionais, estaduais ou nacionais.
